## What is Haze?

**Haze** is an open-source organization building native desktop applications for **Linux** and **Windows**.

We believe great desktop software should be fast, focused, and free — no Electron, no bloat, no compromise.

> 🚧 **We're just getting started.** Our first projects are currently in development. Watch this space.

## Our Principles

- **Native first** — real desktop apps, not web apps in a box
- **Cross-platform** — Linux & Windows without compromise
- **Open source** — always free, forever auditable
- **Focused** — one app, one purpose, done right
